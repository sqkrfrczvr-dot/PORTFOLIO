Ce projet est un portfilio personnel et a été créé dans le cadre d’une candidature à la formation SUPINFO. Sa mise au point a nécessité l'utilisation de HTML et de CSS.

Le site présente :
- une page de présentation
- mes compétences
- mes projets
- ma formation
- un formulaire de contact.

 Technologies utilisées
- HTML5  
- CSS3  
- GitHub Pages (pour la mise en ligne).

Objectif du projet
L’objectif de ce projet est de :
- découvrir le développement web
- apprendre les bases du HTML et du CSS
- utiliser GitHub pour héberger un site web.

 Mise en ligne
Le site est hébergé grâce à GitHub Pages.
